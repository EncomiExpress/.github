# EncomiExpress

> Sistema de gestión web y móvil para OsvaldoC Mensajería y Logística S.A.S.

EncomiExpress es un sistema de gestión web y móvil desarrollado para OsvaldoC Mensajería
y Logística S.A.S., empresa antioqueña especializada en el transporte de encomiendas con
cobertura principal en el Bajo Cauca y Risaralda. Antes de su implementación, la operación
dependía de WhatsApp, un sistema básico de guías en PDF llamado Beewo, y hojas de cálculo
para consolidar pagos, lo que generaba duplicidad de información, errores en digitación
y falta de trazabilidad.

EncomiExpress centraliza en una sola plataforma la gestión de ventas, encomiendas, rutas,
conductores, vehículos y pagos, reemplazando los procesos manuales por un sistema digital
que facilita el control operativo y la toma de decisiones administrativas. Desarrollado
como proyecto de grado del programa Tecnología en Análisis y Desarrollo de Software — SENA,
Regional Antioquia.
